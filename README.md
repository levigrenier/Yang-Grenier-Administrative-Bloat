# Yang-Grenier-Administrative-Bloat
A repository containing the replication materials and code for the paper "What Leads to Administrative Bloat? A Dynamic Model of Administrative Cost and Waste" by Vicky C. Yang and Levi Grenier.

The code constituting the model and its analysis are provided in the files beginning with "Functions_". The code necessary to reproduce the figures is provided in the files beginning with "Fig_". The folder structure for the saved data should be maintained for proper reading from the scripts.

Please reach out to the corresponding author with any inquiries.
